🚕 Política de Privacidad - Taxi Colombia

Última actualización: Enero 2026

1. Información que Recopilamos

Taxi Colombia recopila la siguiente información:

  • Conductores: Nombre, número de teléfono, placa del vehículo
  • Clientes: Número de teléfono, nombre (si lo proporciona), ubicación de recogida
  • Ubicación GPS: Para mostrar servicios cercanos y calcular tiempos de llegada

2. Cómo Usamos la Información

Utilizamos la información recopilada para:

  • Conectar clientes con conductores disponibles
  • Calcular distancias y tiempos estimados de llegada
  • Permitir la comunicación entre cliente y conductor
  • Mejorar nuestros servicios

3. Compartición de Datos

Solo compartimos información personal en los siguientes casos:

  • El número y dirección del cliente se comparte únicamente con el conductor que acepta el servicio
  • No vendemos ni compartimos datos con terceros para fines publicitarios

4. Ubicación GPS

La aplicación utiliza servicios de ubicación para:

  • Mostrar servicios ordenados por cercanía
  • Calcular tiempo estimado de llegada
  • Facilitar la navegación hacia el cliente

Puedes desactivar el acceso a la ubicación en la configuración de tu dispositivo, aunque esto limitará algunas funciones.

5. Seguridad de los Datos

Implementamos medidas de seguridad para proteger tu información:

  • Conexiones encriptadas (HTTPS)
  • Acceso restringido a bases de datos
  • Eliminación de datos de servicios completados después de 30 días

6. Tus Derechos (Ley 1581 de 2012 - Colombia)

De acuerdo con la ley colombiana de protección de datos, tienes derecho a:

  • Conocer qué datos tenemos sobre ti
  • Actualizar o corregir tu información
  • Solicitar la eliminación de tus datos
  • Revocar tu autorización para el tratamiento de datos
